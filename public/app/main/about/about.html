About

What is SlackInspector?

SlackInspector is an open-source web application which allows you to perform a linguistic analysis of messages from specific Slack channel with IBM Watson Tone Analyzer. Analysis will help to detect emotional tones, social propensities, and writing styles in written communication.

How it works

SlackInspector consumes Slack API to fetch the messages from selected channel from authorized Slack team and IBM Watson Tone Analyzer API to perform a linguistic analysis of fetched messages.

Slack Authorization

In order to perform a linguistic analysis you need to authorize SlackInspector application with Slack. App requires permission to get a list of channels from authorized team and access channel message history. Requested OAuth scopes are channels:read and channels:history. You can read more about Slack API OAuth Scopes here.

Privacy

SlackInspector does not collect or store any sensitive information. Your unique Slack token which is used for authorization is encrypted and stored in your browser's local storage. You have an option to share your anonymous analysis results with SlackInspector. Only tone scores will be collected. They are used in average analysis results report.
